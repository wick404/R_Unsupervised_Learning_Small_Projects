##Introduction

The goal of my dimension reduction project is to optimize the analysis of diabates dataset collected from the Iraqi society.

The dataset contains essential medical laboratory data such as Blood Sugar Level, Age, Gender, Creatinine ratio (Cr), BMI, Urea, Cholesterol (Chol), Fasting lipid profile, HBA1C, etc.

Next I am cleaning the data, since in the Gender column some of the records were stored as lowercase letters, I make sure that everything is converted into uppercase. In case of CLASS, some of our data contained unnecessary whitespaces.

diabetes <- diabetes %>%
  select(-ID, -No_Pation,)

unique(diabetes$Gender)
## [1] "F" "M" "f"
diabetes$Gender <- toupper(diabetes$Gender)
#convert male to 1 female to 0
diabetes$Gender <- ifelse(diabetes$Gender == 'M', 1, 0)

unique(diabetes$CLASS)
## [1] "N"  "N " "P"  "Y"  "Y "
diabetes$CLASS <- trimws(diabetes$CLASS)
#N = 1, P = 2, Y = 3
diabetes$CLASS <- as.numeric(factor(diabetes$CLASS, levels = c("N", "P", "Y")))

sum(is.na(diabetes))
## [1] 0

Next, I am using the KMO test and the Bartlett test to see if our data is suitable for factor analysis. Since our overall MSA score is above 0.6, and our p-value = 0, our data is suitable, therefore we can compute PCA (Principal Component Analysis).

I am using Kaisers Stopping Rule and the elbow method to determined the optimal number of components. Whichever components eigenvalue is higher than 1 will be considered as a component. Based on the above mentioned criteria, we can say that we will use 4 components.

pca <- prcomp(diabetes, center = TRUE, scale = TRUE)

eig.val <- get_eigenvalue(pca)

Summary: Thanks to dimension reduction, we could reduce our 10 dimensions to 4 while preserving almost 60% of the variance of the original dataset
